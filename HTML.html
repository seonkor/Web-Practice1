HTML

HyperText Markup Language (HTML)는 웹 페이지와 웹 애플리케이션을 만들기 위한 기초적인 마크업 언어이다.
웹 브라우저는 웹 서버와 로컬 저장소로부터 HTML 문서를 받아 멀티미디어 웹 페이지에 구성한다.
HTML은 웹 페이지의 구조를 의미론적으로 표현하고 원래 문서의 모양에 대한 단서를 포함한다.

HTML 요소는 페이지의 구성 요소이다. HTML 구조를 사용하면, 이미지 및 대화형의 양식과 같은 기타 개체가 렌더링 된 페이지에 포함될 수 있다.
제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 표시하여 구조화 된 문서를 작성하는 수단을 제공한다.
HTML 요고는 꺾쇠 괄호를 사용하여 작성된 태그로 표시된다.
