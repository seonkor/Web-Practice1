WEB

World Wide Web (WWW 또는 웹)은 문서 및 기타 웹 리소스가 URL (Uniform Resource Locator)로 식별되고 하이퍼 텍스트 링크로 상호 연결되며 인터넷을 통해 액세스 할 수있는 정보 공간이다.
영국 과학자 Tim Berners-Lee는 1989년에 World Wide Web을 발명했다. 그는 1990년 스위스 CERN에 근무하면서 최초의 웹 브라우저 컴퓨터 프로그램을 작성했다.
웹 브라우저는 1991년 CERN 외부에서 출시되었으며, 1991년 1월부터 다른 연구 기관에 처음으로, 1991년 8월 인터넷을 통해 일반 대중에게 공개되었다.
